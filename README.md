# Application de gestion de bibliothèque

Projet de fin de formation - Concepteur Développeur d'Applications (Studi)

## Stack technique
- React 18 + Vite
- Tailwind CSS
- React Router v6
- Axios + JWT
- React Hook Form + Zod

## Installation
\`\`\`bash
npm install
cp .env.example .env
npm run dev
\`\`\`

## Fonctionnalités
- Authentification sécurisée (JWT)
- Gestion du catalogue (CRUD livres)
- Gestion des adhérents
- Suivi des emprunts/retours
- Tableau de bord statistiques
